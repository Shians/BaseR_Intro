Preface

This introduction to base R course was designed for biologists with no previous programming experience. It has a focus on analysing RNA-sequencing data using the Bioconductor package limma. The F1000 paper ‘RNA-seq analysis is easy as 1-2-3 with limma, Glimma and edgeR’ was used as a guide for the latter sessions.

It is intended to be delivered over four 2.5 hour workshops at the Walter and Eliza Hall Institute of Medical Research (WEHI).
